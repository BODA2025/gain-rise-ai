# Components

Organized by domain to keep the app scalable.

- `common/` — cross-cutting building blocks (EmptyState, LanguageSwitcher, ThemeToggle).
- `layout/` — page shells, containers.
- `navigation/` — nav-specific bits (reserved).
- `site/` — marketing/site chrome (SiteNav, SiteFooter, Section, PageHeader).
- `ai/` — AI-facing UI (prompt forms, streaming panels). Reserved for future modules.
- `nutrition/` — macro rings, food cards, meal plan tables.
- `workout/` — exercise cards, session runners.
- `dashboard/` — widgets (StatWidget, charts, goal cards).
- `ui/` — shadcn primitives. Do not edit by hand.
